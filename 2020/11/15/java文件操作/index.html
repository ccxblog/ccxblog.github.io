Java IO

主要内容

  • Java.io.File 类的使用

  • IO原理及流的分类

    • 文件流：FileInputStream , FileOutputStream , FileReader , FileWriter 数据流的读写都是基于文件的操作。

    • 缓存流：BufferedInputStream , BufferedOutputStream , BufferedReader , BufferedWriter数据流的读写都是基于内存的操作。

    • 转换流：InputStreamReader , OutputStreamWriter

File 类

  • java.io.File类 ： 文件和目录路径名的抽象表示形式，与平台无关。

  • File 能新建 ， 删除 ， 重命名文件和目录， 但 File 不能访问文件内容本身。 如果需要访问文件内容本身，则需要使用输入 / 输出流。

  • File 对象可以作为参数传递给流的构造函数。

  • File 的常见构造方法：

    • public File (String pathname) 以 pathname 为路径创建 File 对象 ， 可以是绝对路径或者相对路径， 如果 pathname 是相对路径 ， 则默认的当前路径在系统属性 use.dir 存储。
    • public File (String parent , String child ) 以 parent 为父路径 ， child 为子路径创建File 对象。
package main;

import java.io.File;
import java.io.IOException;

public class Application {
    public static void main(String[] args) {
        File f = new File("D:\\Typora\\1.txt");
        // File f1 = new File("D:\\Typora" , "1.txt"); 与上面等价 ， 使用少
        
        System.out.println(f.getName()); // 获取文件名  1.txt
        System.out.println(f.getPath()); // 获取当前的目录 D:\Typora\1.txt ，new 的时候写的路径

        File f1 = new File("src\\main\\java\\main");
        System.out.println(f1.getAbsolutePath()); // 获取当前文件的绝对路径 E:\fp\MCMS\src\main\java\main
        // 返回一个用当前文件的相对路径构建的 File 对象
        System.out.println(f1); // src\main\java\main
        // 返回一个用当前文件的绝对路径构建的 File 对象
        System.out.println(f1.getAbsoluteFile()); // E:\fp\MCMS\src\main\java\main
        System.out.println(f1.getParent()); // 返回父级路径 src\main\java


        f.renameTo(new File("D:\\Typora\\2.txt")); // 重命名文件或文件夹
        System.out.println(f.exists()); // 判断文件或者文件夹是否存在 bool
        System.out.println(f.canRead()); // 判断是否可读  bool
        System.out.println(f.canWrite()); // 判断是否可写  bool
        System.out.println(f.isFile()); // 判断是不是文件 bool
        System.out.println(f.isDirectory()); // 判断是不是文件夹 bool
        System.out.println(f.lastModified()); // 获取文件的最后修改时间 ， 返回毫秒数
        System.out.println(f.length()); // 返回文件的长度 ， 单位是字节数


        File f2 = new File("D:\\Typora\\b.txt");

        if(!f2.exists()){ // 如果文件是否存在
            try {
                f2.createNewFile(); // 创建新文件
            }catch (IOException e){
                e.printStackTrace();
            }
        }
        f2.delete(); // 删除文件

        File f3 = new File("D:\\Typora\\abc");

        f3.mkdir(); // 创建目录

        String[] dir = f3.list(); // 返回当前文件夹的子集名称 ， 包括目录 和 文件
        for(String s : dir) System.out.println(s); // 输出当前文件夹里的文件

        File[] flist = f.listFiles(); // 返回当前文件夹的File 对象 ， 包括文件和目录
        if(flist != null ) for(File cf : flist) System.out.println(cf);

        dfs(new File("D:\\test")); // 递归遍历文件夹
    }

    public static void dfs(File f)
    {
        if(f.isFile()) System.out.println(f.getAbsolutePath() + " is file");
        else
        {
            System.out.println(f.getAbsolutePath() + " is files");
            File [] ne = f.listFiles();
            if(ne != null) for(File c : ne) dfs(c);
        }
    }

}

IO

  • io流是用来处理设备之间的数据传输。

  • Java程序中，对于数据的输入 / 输出操作以流（stream）的方式进行。

  • Java.io包提供了各种 ”流“ 类和接口 ， 用以获取不同种类的数据，并通过标准的方法输入或输出数据。

  • 流的分类

    • 按操作数据单位不同分为: 字节流(8 bit) , 字符流(16 bit)

    • 按数据流的流向 不同分为 ： 输入流 ， 输出流

    • 按流的角色不同分为：节点流 ， 处理流

      （抽象基类） 字节流 字符流
      输入流 InputStream Reader
      输出流 OutputStream Writer

java IO 共涉及 40 多个类， 实际上非常规则，都是从上表的 4 个抽象基类派生的。由这 4 个类派生出来的子类名称都是以其父类名作为子类名后缀。

文件字节输入输出流

文件字节输入流：FileInputStream

文件字节输出流：FileOutputStream

文件字节流可以操作字符的文档，还可以操作其它(图片 ， 压缩包等)；

package main;
import java.io.File;
import java.io.FileOutputStream;
import java.lang.Byte;
import java.io.FileInputStream;
import java.io.FileNotFoundException;

public class Application {
    /**
     * 文件字节输入流
     */
    public static void testFileInputStream(){
        try {
            FileInputStream in = new FileInputStream("D:\\test\\a\\d.txt");
            byte[] cin = new byte[10]; // 设置一个 Byte 数组接收文件的内容
            in.read(cin); // 返回读取的长度 ， 返回 -1 结束
            System.out.println(new String(cin));
            in.close(); // 关闭文件
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    /**
     * 文件字节输出流
     */
    public static void testFileOutputStream(){
        try {
            FileOutputStream out = new FileOutputStream("D:\\test\\a\\d.txt");
            String s = "abscderf";
            out.write(s.getBytes());  // 将字符串转换为 Bytes 类型 写到内存
            out.flush(); // 将内存数据刷到硬盘
            out.close();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }

    public static void main(String[] args) {
        testFileInputStream();
        testFileOutputStream();
    }
}

文件字符输入输出流

文件字符输入流

读取文件步骤：

  1. 建立一个流对象，将已存在的一个文件加载进流。
  2. 创建一个临时存放数据的整数或数组
  3. 调用流对象的读取方法将流中的数据读取到数组中

一个字符一个字符读


import org.junit.jupiter.api.Test;

import java.io.File;
import java.io.FileReader;
import java.io.IOException;

public class TestFileInputStream {
    @Test
    public void test(){
        //hello.txt
        File f = new File("hello.txt"); // File类的实例化
        FileReader fr = null; // 提权，让 fr 在 finally 也可以用
        try {

            fr = new FileReader(f); // 字符流的实例化
            int data;
            while((data = fr.read()) != -1){ // 读取操作
                System.out.print((char)data);
            }

        } catch (IOException e) {

            e.printStackTrace();

        } finally {

            if(fr != null) {
                try {
                    fr.close(); // 关闭
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }

        }

    }
}

文件字符输出流

写文件步骤：

  1. 建立对象 ， 如果文件不存在，则自动创建文件，如果存在，如果使用 FileWriter(file) / FileWriter(File , false)则会对原有文件进行覆盖 ， 如果使用 FileWriter(file , true) 则会在原来文件的末尾追加。
  2. 创建要输出的字符串数组
  3. 调用流对象的写方法，将数据写到文件内 
@Test
    public void test3(){
        File f = new File("hello2.txt");

        FileWriter fw = null;

        try {
            fw = new FileWriter(f); // 覆盖方式写入
            fw.write("I have a dream!");
            //append 参数为 true 追加写入
            fw = new FileWriter(f , true);
            fw.write("you need have a dream!");
        } catch (IOException e) {
            e.printStackTrace();
        }finally {
            try {
                if(fw != null)
                fw.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }
    }

转换流

转换流提供了，字符流和字节流之间的转换

InputStreamReader:将一个字节的输出流转换为字符的输出流 。把看不懂的字节【数组】，转换为看得懂的字符串，即解码。

OutputStreamWriter:将一个字符的输出流转换成字节的输出流。把看得懂的字符转换成看不懂的字节，即编码。

  • 使用转换流，将UTP-8格式文件，转为GBK文件
@Test
    public void InputStreamReaderAndOutputStreamWriter() throws IOException {

        FileInputStream fis = new FileInputStream("hello.txt");
        FileOutputStream fos = new FileOutputStream("hello1.txt");
        InputStreamReader isp = new InputStreamReader(fis , "UTF-8");
        OutputStreamWriter osw = new OutputStreamWriter(fos , "GBK");

        char c[] = new char[1024];
        int len;
        while((len = isp.read(c)) != - 1){
            osw.write(c , 0 , len);
        }

        isp.close();
        osw.close();
    }

对象流

ObjectInputStream , ObjectOutputStream

作用：用于存储和读取基本数据类型或对象的处理流，它的强大处在于可以把 java 中的对象写到数据源中，也能把对象从数据源中还原回来。

序列化：将内存中的java对象保存在磁盘中或者通过网络进行传输，使用ObjectOutputStream实现

反序列化：将磁盘文件中的对象还原为内存中的 java 对象 ， 使用 ObjectInputStream实现

自定义类要想序列化 一般要实现 Serializable接口 ， 然后提供一个全局常量

static final long serialVersionUID 并设置值。还得保证类中的属性也是可以序列化的。

补充：ObjectInputStream 和 ObjectOutputStream是不可以操作 static 和 transient修饰的成员的

  • 创建实现了可序列化的类 Person
import java.io.Serializable;

public class Person implements Serializable {
    private int age;
    private String name;
    static final long serialVersionUID = 2423579873247L;
    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    @Override
    public String toString() {
        return "Person{" +
                "age=" + age +
                ", name='" + name + '\'' +
                '}';
    }
}
