什么是数据库

  • 数据库（DB ， DataBase）
  • 概念：数据仓库 ， 软件 ，安装在操作系统上，可以存储大量的数据
  • 作用：存储数据 ， 管理数据

数据库的列类型

  1. 数值

    tinyint 十分小的数据 1个字节
    smallint 较小的数据 2个字节
    mediumint 中等大小的数据 3个字节
    int 标准的整数 4个字节
    bigint 较大的数据 8个字节
    float 单精度 4个字节
    double 双精度 8个字节
    decimal 字符串形式的浮点数 金融计算的时候

  1. 字符串

    char 固定大小字符串 0-255
    varchar 可变字符串 0-65535
    tinytext 微型文本 0 - 2^8 - 1
    text 文本串 0 - 2^16-1

  1. 时间日期

    Java有java.util.Data类

    date YYYY-MM-DD 日期
    time hh:mm:ss 时间
    datetime YYYY-MM-DD hh:mm:ss 最常用的时间格式
    timestamp 时间戳 1970.1.1到现在的毫秒数
    year 年份表示

  1. null

    没有值未知 ， 不用使用 null 进行计算

数据库的字段属性

unsigned :

  • 无符号整数
  • 声明了该列不能为负数

zerofill:

  • 0 填充的
  • 不足的位数用 0 填充 。 int(3) 5 —> 005

直增:

  • 顾名思义 ， 默认在上一条的记录上 + 1
  • 通常用来设计唯一的主键 ， 必须是整数类型
  • 可以自定义设置主键自增的起始值和步长

非空: NULL not null

  • 假设设置为 非空 不给它写值 就会报错

默认:

  • 设置默认值

操作数据库(了解)

  • MySql 数据课不区分大小写
  1. 创建数据库
CREATE DATABASE [IF NOT EXISTS] work;
  1. 删除数据库
DROP DATABASE IF EXISTS WORK;
  1. 使用数据库
USE school;  -- 切换到 school数据库

如果表名或者特殊字段是 sql 关键字 ， 可以在两边 加上 `` 反引号

  1. 查看数据库
SHOW DATABASES

创建数据库表

  • 目标创建一个表，学号 int , 登录密码varchar(20) , 姓名 ， 性别 varchar(2) , 出生日期datatime
  • 注意点使用英文符号 ， 表的名字和字段尽量用 ``括起来
  • AUTO_INCREMENT 直增
  • 字符串用 单引号括起来
  • 除了最后一个语句 ， 其它语句结束后加上，
  • PRIMARY_KEY 主键 ， 一般在最后声明
CREATE TABLE `student`(
   `id` INT(4) NOT NULL AUTO_INCREMENT COMMENT '学号', 
   `name` VARCHAR(30) NOT NULL DEFAULT '匿名' COMMENT '姓名',
   `pwd` VARCHAR(30) NOT NULL DEFAULT '123456' COMMENT '密码',
   `sex` VARCHAR(2) NULL DEFAULT '男' COMMENT '性别',
   `birthday` DATETIME NULL COMMENT '出生时间',
   PRIMARY KEY(`id`)
)ENGINE = INNODB DEFAULT CHARSET=utf8

格式

CREATE TABLE [IF NOT EXISTS] '表名'(
    '字段名' 列类型 [属性] [索引] [注释] ,
    '字段名' 列类型 [属性] [索引] [注释] ,
    ...
    '字段名' 列类型 [属性] [索引] [注释] ,
)[表的类型][字符集设置][注释]

DML语言

数据库的意义 ：数据存储 ， 数据管理

DML语言：数据操作语言

  • insert 插入
  • update
  • delete

添加

插入语句 insert into 表名 ([字段名1] ,[字段名2],[字段名3]) values(‘值1’) ,(值2),(值3)

插入多个字段：

INSERT INTO stu (`name` , `age`) VALUES ('sb' , 18) , ('hs' , 19)

注意事项：

  1. 字段和字段之间使用英文逗号隔开
  2. 字段是可以省略的，但后面的值必须要一一对应，不能少
  3. 可以同时插入多条数据 ， values 后面的值 ，需要使用 ， 隔开即可 (),()

修改

update 修改谁 (条件) set 表中字段名 = value where 条件

修改表某行的信息

UPDATE `stu` SET `name` = 'xi' WHERE id = 1; -- 把stu表 中 id = 1的行中name字段的值设为xi

如果没有后面的条件

UPDATE `stu` SET `name` = 'xi' -- 把全部name字段的值设为xi

修改多个属性，用 ， 隔开

UPDATE `stu` SET `name` = 'xi' ,age = 99 WHERE id = 1;

条件：where 子句 运算符 id 大于 ， 等于 ， 在某个区间修改… 操作符会返回bool值

操作符 含义 范围 结果
= 等于 5 = 6 false
<> 或 != 不等于 5 != 6 true
>= 大于等于 5 >= 6 true
<= 小于等于 5 <= 6 true
BETWEEN…and… 在某个范围内 [2,5]
and && 5>1 and 1>2 false
or || 5>1 and 1 > 1 true

删除

delete 命令

语法 `delete from 表名 [while 条件]

---- 删除数据 , 避免这样写 ， 会全部删除
DELETE FROM 'student'
-- 
DELETE FROM 'student' WHERE id = 1;

TRUNCATE (truncate) 命令

作用：完全清空一个数据库表， 表的结构和索引约束不会变！

-- 清空 student 表
TRUNCATE 'student'

delete 和 truncat 的区别：

  • 相同点：都能删除数据，而且都不能删除表结构
  • 不同点：
    • truncat 重新设置自增列 ， 计数器归0
    • truncat 不会影响事务

DQL查询数据

DQL（data query language：数据查询语言）

  • 所有的查询操作都用它 Select

  • 简单的查询和复杂的查询都能做

  • 数据库中最核心的语言，最重要的语句

  • 使用频率最高的语句

查询指定字段

select * from student -- 查询学生表全部学生信息
select name , num from student  -- 查询学生表全部学生的姓名和学号
-- 别名 ， 给结果取个名字，将原本列名换个名字 AS ， 可以给字段和表起别名
select num AS 学号 from student -- 原本列名是num 查询出来显示的列名是 学号

-- 函数 Concat(a , b)
select concat('姓名 ' , name) from student -- 显示的时候name字段都加上 姓名 前缀

语法：select 字段... from 表名

去重 distinct

作用去除 select 查询出来的结果中重复的数据，重复的数据只显示一条

--- 查询下有哪些学生参加了考试
select * from result -- 查询全部的考试成绩
select num from result -- 查询哪些同学参加了考试
-- 有可能一个同学参加了多门考试，会有重复数据 所以去重
select distinct num from result
-- 学员考试成绩加一分查看
select res + 1 from result

where 条件子句

作用：检索数据中 符合条件的 值，搜索的条件由一共或多个表达式组成 ， 结果是 bool 值

运算符 语法 描述
and && a and b — a&& b 逻辑与
or || a or b — a|| b 逻辑或
not ! not a —- !a 逻辑非 
--- 查询成绩在 95 - 99 之间的学生
select num from sturesult 
where s >= 95 and s <= 99

--- 区间模糊查询
select num from sturesult 
where s between 95 and 99 

--- 除了学号1000以外的学生
select num from student
where num != 1000

模糊查询

  • 模糊查询的本质还是比较运算符
运算符 语法 描述
is null a is null 如果操作符为 null ，结果为真
is not null a is not null 如果操作符不为 null 结果为真
between…and a between b and c 若a在 b与 c 之间则结果为真
Like a Like b SQL 匹配，如果a能匹配b 则结果为真
in a in （a1 , a2 ,a3..） 假设a在，ai 内 结果为真 
--查询姓刘的同学的名字
-- Like 结合 % (匹配0 - 任意个字符) _ (匹配一个字符)
select StudentNo from student where Stuname LIKE '刘%'

-- 查询名字姓刘 且后面只有一个字的同学
select StudentNo from student where Stuname LIKE '刘_'

------------------ in ------------
-- 查询 1001 1002 1003号学员的信息
select * from student where studentNo in (1001,1002,1003)

-- 查询在 地址为NULL的学生
select * from student where address is null

联表查询

JOIN

--- 查询学生的学号 ， 姓名 和 成绩
-- 一般成绩不会和姓名信息放一起 ， 所以涉及多表查询

-- inner join
select s.stuNo stuName , stuCore from student AS s
INNER join result AS r
ON s.stuNo = r.stuNo

-- right join
select s.stuNo stuName , stuCore from student AS s
rigth join result AS r
ON s.stuNo = r.stuNo

-- left join
select s.stuNo stuName , stuCore from student AS s
left join result AS r
ON s.stuNo = r.stuNo

-- join ( 连接的表 ) on( 判断的条件 ) 连接查询
-- where 等值查询

--- 多表查询 
select stuNo , stuScore , SutClass 
from student AS s right join result AS r
ON s.stuNo = r.stuNo  -- 到这里得到一张新表
inner join subject AS sub  -- 前面得到的表 继续连接查询
ON r.stujectNo = sub.stujectNo
操作 描述
inner join 如果表中至少有一个匹配，就返回行
left join 即使右边中没有匹配，也会从左表中返回所有的值
right join 即使左表中没有匹配，也会从右表中返回所有的值

自连接

自己的表和自己的表连接 ， 核心： 一张表拆为两张一样的表即可。

-- 一般用于一个表内有树形结构
select a.name , b.name from student AS a , student AS b
where a.id = b.pid

分页和排序

  • 分页： limit

  • 排序：order by

    • 升序：ASC
    • 降序 : DESC

排序

-- 查询成绩大于 90 分的学生 并按降序显示
select stuName , StuNo , StuScore from student
where StuScore > 90
order by StuScore DESC

分页 

-- 语法 limit(查询的起始值(从第几条开始显示) ， 页面大小)
-- 查询成绩大于 90 分的学生 并按降序显示 , 从第0条数据开始，当前页显示5条数据
select stuName , StuNo , StuScore from student
where StuScore > 90
order by StuScore DESC  
limit 0 , 5

子查询

where(这个值是计算出来的) ， 相当于在 where 里面嵌套一个子查询语句

-- 查询高等数学分数大于80的学生的学号和姓名
select stuNo , stuName from student
where stuNo in (select stuNo from result where stuScore > 80 and ClassName = (
    select ClassName from Class where ClassName = '高等数学'
))

  • 聚合函数
函数名称 描述
count() 计数
sun() 求和
avg() 平均值
max() 最大值
min() 最小值 
select count(stuNo) from student -- 获取student 表中的学生数量
select count(*) from student
select count(1) from student
-- count(字段) ， count(*) , count(1) 都能统计表中的数据
-- count(字段) 会忽略所有的null 值
-- count(*) 不会忽略 null 值
-- count(1) 不会忽略 null 值

-- count(*) 和 count(1) 本质是计算行数

select sum(stuScore) from student; -- 将分数求和

-- 查询不同课程的平均分，最高分，最低分 ， 并且平均分大于80的数据 
-- 核心：根据不同的课程分组
select SubjectName , ave(stuScore) AS 平均分, max(stuScore) , min(stuScore)
from result r
inner join subjects sub
ON sub.subjectNo = r.subjectNo
group by subjectNo -- 通过什么排序
HAVING 平均分大于 80 --  分组后要过滤用 having

Select小结

select 基本结构

select 去重 要查询的字段 from 表 (表和字段可以取别名)

Having 过滤分组后的信息 , 和where 一样
order by 通过那个字段排序 （ ASC , DESC）
