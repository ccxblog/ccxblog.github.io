Java的方法与数组

发表于 2020-11-10 更新于 2020-11-14 分类于 java

Java中的方法

什么是方法

  • System.out.println() 其中 System 是一个类 ， 而 out 是 System 的一个对象 ， println() 就是这个对象的方法。

  • Java方法是语句的集合 ， 它们在一起执行一个功能。

    • 方法是解决一类问题的步骤的有序集合。
    • 方法包含于类和对象中。
    • 方法在程序中被创建，在其它地方被引用
  • 设计方法的原则：设计方法的时候最好保存方法的原子性，就是一个方法只完成一个功能，这样有利于后期的扩展。

方法的定义

  • Java 的方法类似于 C++ 的函数 ， 一般一个方法包括以下语法

    修饰符 返回值类型 方法名(参数类型 参数名){
   ...
   方法体
   ...
   return 返回值;
}

方法的重载

  • 重载就是在一个类中 ， 有相同的函数名字 ， 但形参不同的函数。

  • 方法重载的规则：

    • 方法名必须相同。

    • 参数列表必须不同（个数不同 ， 类型不同 ， 参数排列顺序不同等)。

    • 方法的返回类型可以相同也可以不同。

    • 仅仅返回类型不同不足以构成方法重载。

  • 实现理论：方法名相同的时候，编译器会根据调用方法的参数个数，参数类型等逐一匹配，如果匹配失败，则编译器报错。
public class Main {
    // main 方法
    public static void main(String[] args) {

        System.out.println(add(1, 2));
        System.out.println(add(10.2 , 1.3));
    }
    
    // 构成方法重载
    public static int add(int a , int b) {return a + b;}
    public static int add(double a , double b) { return (int)a + (int)b ;}
    
    public  static double add(int a , int b){return a + b;} // 不构成方法重载
    
}

可变参数

  • Java 支持传递同类型的可变参数给一个方法。
  • 在方法声明中，在指定参数类型后加一个省略号(…)。
  • 一个方法中只能指定一个可变参数，它必须是方法的最后一个参数，任何普通的参数都必须在它之前声明。
public class Main {

    public static void main(String[] args) {
        int res = getMax(1,2,3,4,5);
        System.out.println(res);
    }

    public static int getMax(int ...x){ // 可变参数
        int mx = x[0]; // 可用下标访问
        int n = x.length; // 类似于数组
        for(int i : x) mx = Math.max(mx , i); // 遍历
        return mx;
    }
}

静态方法和非静态方法

  • 静态方法

    在方法声明的时候加static 关键字 ， 被static关键字修饰的方法或者变量不需要依赖于对象来进行访问，只要类被加载了，就可以通过类名去进行访问。

    　　“static方法就是没有this的方法。在static方法内部不能调用非静态方法，反过来是可以的。而且可以在没有创建任何对象的前提下，仅仅通过类本身来调用static方法。这实际上正是static方法的主要用途。” —– JAVA编程思想

    一句话来描述就是：方便在没有创建对象的情况下来进行调用（方法/变量）。

  • 非静态方法

    方法声明时没有static 关键字 ， 方法的调用入口 需要用 new 关键字实例化对象后生成。

class Student{
  public static void say(){ // 静态对象
      System.out.println("Say");
  }
  public void sayhellow(){ // 非静态对象
      System.out.println("say , Hellow");
  }
}
public class Main {
    public static void main(String[] args) {
        Student.say(); // 输出Say
        // Student.sayhellow()  不合法
    Student stu = new Student(); // 实例化对象
        stu.sayhellow(); // 输出 say , Hellow
  }
}

Java 数组

什么是数组

  • 数组是相同类型数据的有序集合。
  • 数组描述的是相同类型的若干数据，按照一定的先后次序排列组合而成的。
  • 其中，每一个数据称作一个数组元素，每一个数组元素可以通过下标访问。

数组的声明创建

  • Java数组遵守先声明后使用的原则， 声明语法如下：

    dataType [] name; // 首选方法 
或
dataType name [];

  • Java使用new 操作符创建数组 ：

    dataType [] arr = new dataType [arrSize];

  • 数组的长度 arr.length

import java.sql.SQLOutput;

public class Main {

    public static void main(String[] args) {
        int [] arr = new int [5]; // 声明创建数组,默认值为0
        int n = arr.length; // 获取长度
        // 一般遍历数组
        for(int i = 0 ;i < n ; i ++) System.out.println(arr[i]);
        for(int i : arr) System.out.println(i); // for-each 遍历数组
    }
}

数组的初始化

  • 静态初始化 ， 这样创建的数组空间是固定的。

    int a [] = {1,2,3};   
Man[] mans = {new Man() , new Man()};

  • 动态初始化 : 包含默认初始化。

    int [] a = new int [5];

多维数组

  • 多维数组可以看成是数组中的数组 ， 比如二维数组可以解释为一个一维数组内的每个元素为一个一维数组。

  • 二维数组定义

int [][] a = new int [2][5] 定义一个2行5列的二维数组。

public class Main {
    public static void main(String[] args) {
        // 静态初始化 ，一共2行 , 第一行的长度为2 ， 第2行长度为5；
        int [][] arr = {{1 , 2} , new int [5]};
        int n = arr.length; int m = arr[1].length; // n = 2 , m = 5;
        System.out.println(arr[0][1]); // 输出 2
        
    }
}

Arrays类

数组的工具类java.util.Arrays

  • Arrays此类包含用来操作数组的各种方法，比如排序和搜索。

    import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        int [] a = {1,3,5,67,34,51,52,5,4,3};
        // 打印数组元素
        System.out.println(Arrays.toString(a));
        Arrays.sort(a); // 排序
        System.out.println(Arrays.toString(a));

        Arrays.fill(a , 0); // 把a数组所有元素的值赋为0
        Arrays.fill(a , 2 , 4 , 0); // 下标从2到4填充为0
        
    }
}
