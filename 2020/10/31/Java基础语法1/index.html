注释

Java中有三种注释

  • 单行注释 : // 注释一行文字

    public class Helloworld {
    public static void main(String[] args){
        // 单行注释
        // 输出一个Hello World
        System.out.println("hello world");
    }
}

  • 多行注释 ： /* 开始然后中间一段文本 */结束

    public class Helloworld {
    public static void main(String[] args){
        /* 注释
           注释
           注释
           注释
        */
        System.out.println("hello world");
    }
}

  • JavaDoc：文档注释

/** 开头然后回车

/**
 * 注释
 * @Description xixi
 * @author xixi
 */
public class Helloworld {
    public static void main(String[] args){
        /* 注释
           注释
           注释
           注释
        */
        System.out.println("hello world");
    }
}

标识符

Java所以的组成部分的名字。类名、变量名以及方法名都被称为标识符。

关键字

标识符

标识符命名遵守命名规范

  • 以字母下划线火$开头。
  • 首字母后可以是字母、下划线、数字或者$的任意字符组合。
  • 不能使用关键字作为标识符。
  • Java可以使用使用中文命名，当不建议这样去使用

数据类型

  • 强类型语言

    要求变量的使用严格符合规定，所有变量必须先定义后使用。列如 Java中 int类型不能赋值为字符串。

  • 弱类型语言

    要求变量的使用符合规定即可。列如：vb，js。

  • Java的数据类型分为两大类

    • 基本数据类型
    • 引用数据类型

类型转换

  • 由于Java是强类型语言，所以进行某些运算的时候需要进行类型转换。
低--------------------------------------------->高
byte,char,short -> int -> long -> float -> double
  • 运算中不同类型先转为同一类型，再进行运算。

强制类型转换

类型从高到低需要进行强制类型转换

public class Main {
    public static void main(String[] args) {

        int a = 120;
        // byte b = a; 不允许
        byte b = (byte)a;
        System.out.println(b);
        
    }
}

自动类型转换

类型从低到高不用进行强制类型转换。

类型转换注意点

  1. 不能对布尔类型进行转换。
  2. 不能把对象类型转换为不相干类型
  3. 把高容量转为低容量时，要强制转换
  4. 转换的时候可能会出现内存溢出，或者精度问题
public class Main {
    public static void main(String[] args) {

        int d = 128;
        byte by = (byte)d;
        System.out.println(by); // 发生溢出 ， 输出-128

        double a = 23.4;
        float  b = -11.56f;
        System.out.println((int)a); // 输出23
        System.out.println((int)b); // 输出 -11
    }
}

变量

  • 变量是可以变化的量

  • Java是一种强类型语语言，每个变量都得声明其类型。

  • Java 变量是程序中最基本的存储单元，其要素包括变量名，变量类型和作用域。

        type varname [= value ] [{, varname [= value]}];
// 数据类型 变量名 = 值; 可以，隔开声明多个变量
  • 注意事项
    • 每个变量都有类型，类型可以是基本类型，也可以是引用类型。
    • 变量名必须是合法的标识符。
    • 变量声明是一条完整的语句，最后记得上分号。

常量

常量是初始化后不能改变的值，常量声明的关键字final，一般格式：

final 变量名 = 值

变量的命名规范

  • 所有的变量，方法，类名：见名知意。
  • 类成员变量：首字母小写和驼峰原则（除了第一个单词外，后面的单词首字母大写）–monthSalay。
  • 局部变量：首字母小写和驼峰原则。
  • 常量：大写字母和下划线。
  • 类名：首字母大写和驼峰原则。
  • 方法名：首字母小写和驼峰原则。
