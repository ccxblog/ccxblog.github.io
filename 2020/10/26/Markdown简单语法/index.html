Markdown学习

标题

一个#加上空格 加上标题

一级标题 # 
二级标题 ##
三级标题 ###
四级标题 ####
五级标题 #####
六级标题 ######

列表

无序列表可以以-开头 然后加列表名

  • 列表1
  • 列表2
    有序列表把-替换成数字或者字母
    a. 字母列表
    b. 字母列表
  1. 数字列表
  2. 数字列表

超链接

一对中括号[]里面填 显示内容，后面跟对小括号，小括号里面是url

[名字](路径)

百度

插入图片

超链接中括号前面加个！号

!(名字)[路径]

字体

文字左右两边各一个星号，则中间的文字为斜体
文字左右两边各两个星号，则中间的文字为加粗
加粗加斜体就三个星号

两边波浪号加上删除线

代码的插入

两个反引号中间加代码（tab）上面那个键,插入一行
#include <iostream>

多行为开始三个反引号，结束3个反引号

#include <iostream>

using namespace std;

int main()
{

    return 0;
}

引用

一个大于号后空格跟引用

线条

三个—或者三个*加分割线

表格

在Typora右击可以直接插入表格

可以打出以下格式，然后在源代码部分把中间的换行去掉

|名字|性别|生日|

|–|–|–|

|张三|男|1997.1.1|

名字 性别 生日
张三 男 1997.1.1
