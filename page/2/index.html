  • 什么是 JavaWeb

JavaWeb 是指，所有通过 Java 语言编写可以通过浏览器访问的程序的总称，叫 JavaWeb。

请求是指客户端给服务器发送数据，叫请求 Request。

  • 什么是响应

响应是指服务器给客户端回传数据，叫响应 Response

  • 请求和响应的关系

请求和响应是成对出现的，有请求就有响应。

字符串

String

  • String 是声明为 final 的 ， 所以不可被继承

  • String实现了 Serializable 接口（可序列化接口）：所以String支持序列化

    • 实现了 Comparable 接口 ， 表示 String 可以比较大小

  • String 内部 定义了 final 的 char 类型数组 value 用于存储字符串数据 ， 所以数组不能被重新赋值，且数组的元素也不能被修改。

所以 String 代表一个不可变的字符序列 , 当对 String 对象进行赋值 或 进行连接操作时 需要重新指定内存区域赋值 ， 不能使用原有的 value 进行赋值

## 多线程

程序：是为了完成特定任务，用某种编程语言编写的一组指令集合。是一段静态的代码

进程：是程序的一次执行过程，或是正在运行的一个程序。是一个动态的过程。它有自身产生，存在，消亡的过程——————生命周期，进程是系统资源分配的最小单位。

线程：进程可进一步细化为线程，是一个程序内部的执行路径，线程作为调度和执行的最小单位，每个线程拥有独立的运行栈和程序计数器 PC，线程的切换开销小。
