Java中的方法

什么是方法

  • System.out.println() 其中 System 是一个类 ， 而 out 是 System 的一个对象 ， println() 就是这个对象的方法。

  • Java方法是语句的集合 ， 它们在一起执行一个功能。

    • 方法是解决一类问题的步骤的有序集合。
    • 方法包含于类和对象中。
    • 方法在程序中被创建，在其它地方被引用
  • 设计方法的原则：设计方法的时候最好保存方法的原子性，就是一个方法只完成一个功能，这样有利于后期的扩展。

注释

Java中有三种注释

  • 单行注释 : // 注释一行文字

    public class Helloworld {
    public static void main(String[] args){
        // 单行注释
        // 输出一个Hello World
        System.out.println("hello world");
    }
}

  • 多行注释 ： /* 开始然后中间一段文本 */结束

    public class Helloworld {
    public static void main(String[] args){
        /* 注释
           注释
           注释
           注释
        */
        System.out.println("hello world");
    }
}

Java的特性与优势

简单性

Java可以说是c/c++的纯净版本，它没有头文件，没有指针，也不用太在意内存管理，学习容易。

面向对象

面向对象将重点放在对象以及对象的接口上，面向对象更符合人的思维。

可移植性

Java的代码可以跨平台的移植，（write once , ran anywhere）写一次可以到处运行。就是说在Windows上写的程序可以在Linux上运行。
